# wx_pay_to_df_beancount

#### Description
**wx_pay_to_df_beancount** 是一个开源 Python 工具，旨在简化微信支付账单的数据处理流程。本仓库提供了一站式的解决方案，帮助用户将微信支付账单数据高效地转换为 DataFrame 格式，进而再转换为 Beancount 记账格式。
